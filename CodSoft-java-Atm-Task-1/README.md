# ATM Interface – Java (CodSoft Internship – Task 1)

This project is a simple console-based ATM Interface developed as part of the CodSoft Java Internship tasks.  
It demonstrates the use of core Java and basic object-oriented programming concepts.

---

## Project Description

The ATM Interface allows a user to perform basic banking operations such as:
- Depositing money
- Withdrawing money
- Checking account balance
- Exiting the application

The application works through a menu-driven console interface and focuses on clean logic and proper class design.

---

## Technologies Used

- Java
- OOP Concepts
- Scanner class for user input

---

## Project Structure

- `BankAccount`  
  Handles account balance and transaction logic.

- `ATM`  
  Manages user interaction and menu operations.

- `Main`  
  Entry point of the application.

---

## How to Run the Program

1. Clone or download the repository  
2. Open the project in any Java-supported IDE  
3. Compile all Java files  
4. Run the `Main.java` file  

---

## Learning Outcome

- Understanding of class interaction  
- Use of encapsulation  
- Menu-driven program design  
- Input validation
